Module loglan_core.addons.definition_selector

This module provides the DefinitionSelector class which is a Selector for the BaseDefinition object.

The DefinitionSelector class allows for querying and filtering of BaseDefinition objects based on various parameters such as event, key, and language. It is a subclass of the Select class.

class DefinitionSelector (model: Type[BaseDefinition] = loglan_core.definition.BaseDefinition, is_sqlite: bool = False, case_sensitive: bool = False, disable_model_check: bool = False)

This class is a selector model for the BaseDefinition object. It allows for querying and filtering of BaseDefinition objects based on various parameters such as event, key, and language. It is a subclass of the Select class.
