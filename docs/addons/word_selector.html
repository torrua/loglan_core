Module loglan_core.addons.word_selector

This module provides a mechanism to extract words from a database based on various criteria such as event, key, type, and name through the WordSelector class.

Classes

class WordSelector (model: Type[BaseWord] = loglan_core.word.BaseWord, is_sqlite: bool = False, case_sensitive: bool = False, disable_model_check: bool = False)

Class to extract words from a database based on various criteria.

Extends the SQLAlchemy Select class to provide additional functionality.

Initializes the WordSelector object with the provided parameters.

The class to be used as the base key. Must be a subclass of BaseWord.

If SQLite is being used. Defaults to False.
case_sensitive : bool
If the queries should be case-sensitive.

If the model check is disabled during initialization.
